Projects

Rap Rising

Rap Rising is a web application that keeps track of new up and coming rappers in the current industry. The rappers are ranked based of current popularity and the amount of music stream they get. As a rapper gains more popularity and music streams, he or she will move up in the list. This web application created so users can discover new artist's and new music. The website also displays some basic information about the rapper such as, name, age, and their top songs. This project was made using HTML, CSS, and JavaScript.
